# 🎨 Creative Playground Lab

Creative Playground Lab is an **interactive creative coding playground** that showcases a collection of **visual simulations, generative art experiments, and physics-inspired animations** built using modern web technologies.

The project demonstrates how **mathematics, physics, and creative coding** can be combined to produce **beautiful interactive visual experiences directly in the browser**.

Users can explore different simulations, adjust parameters in real time, and observe how particles, forces, and generative algorithms create evolving visual patterns.

---

# ✨ Features

- 🌌 **Interactive Visual Simulations**
- ⚛ **Real-time Particle Systems**
- 🎛 **Live Parameter Controls**
- 🌀 **Mathematical & Physics Inspired Experiments**
- ✨ **Smooth WebGL Rendering**
- 🖱 **Interactive Camera Controls**
- 📷 **Screenshot Capture**
- ⏸ **Pause / Resume Simulations**
- 🎨 **Creative Coding Experiments**

# 🧠 Experiments Included

Creative Playground Lab contains multiple visual experiments such as:

- Particle motion systems  
- Physics-based interactions  
- Generative mathematical patterns  
- Visual attractor simulations  
- Interactive animation systems  
- Dynamic particle swarms  
- Real-time visual effects  

Each module explores different concepts from **creative coding, generative art, and simulation design**.
